Are you unsure about your gender? Do you want to know how it would feel to be called something else?

Then this Pronoun Closet is for you! This program inserts your tryout pronouns and name into example sentences. Simply choose the pronoun set(s) you want (or none if you select that option), your preferred name(s) and click on Reload. You also have options for when the program randomly picks a name or pronoun set (per word, sentence, or block), and the option to use the names associated with pronoun sets.

Editing sentence blocks

You can edit or remove the sentence blocks by using the button controls on the bottom right of a block. The format that is used when editing, is explained at the bottom of the page. Use the button at the bottom of the list to add a new sentence block and start editing it.

Sentence Block Edit Format

Pronouns and names are inserted in a sentence block by replacing certain keywords surrounded by curly braces ({}), for example {obj}. If the first letter between the braces is capitalised, the replaced word is also capitalised. The keywords for pronoun conjugations and names can be found in the table below.

Pronoun/Name Keyword
Name {name}
Subject {sub}
Object {obj}
Possessive Determiner {pod}
Possessive Pronoun {pop}
Reflexive {ref}
